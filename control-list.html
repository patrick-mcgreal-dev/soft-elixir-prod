Soft Elixir Control List

GENERAL

  • Navigate between control regions
    [Q] + [arrows]
  • Open track menu
    [esc]

TRACK PROPERTIES

  • Navigate
    [arrows]
  • Edit value
    [F] + [arrows]

SAMPLE BROWSER

  • Navigate
    [arrows]
  • Open sample select
    [F]
  • Delete sample
    [E]

TRACK SEQUENCER

  • Play/pause track
    [space]
  • Mute pattern
    [M] + [number]
  • Solo pattern
    [N] + [number]
  • Navigate
    [arrows]
  • Jump to end
    [A] + [arrows]
  • Enter value
    [F]
  • Edit value
    [F] + [arrows]
  • Create pattern with in next empty slot
    [F] + [J]
  • Copy pattern to next empty slot
    [F] + [K]
  • Delete pattern
    [E]
  • Delete whole step
    [E] + [arrow right]
  • Cut
    [X]
  • Copy
    [C]
  • Paste
    [V]
  • Undo
    [Z]
  • Redo
    [Y]
  • Toggle grab mode
    [G]
  • Create loop (in grab mode)
    [arrows] to select loop region, [space] to play/pause
  • Clear loop
    [G] + [E]

PATTERN SEQUENCER

  • Play/pause pattern
    [space]
  • Mute pattern
    [M] + [number]
  • Solo pattern
    [N] + [number]
  • Preview row
    [R] + [arrow up] / [arrow down]
  • Navigate
    [arrows]
  • Jump to end
    [A] + [arrows]
  • Jump between patterns
    [S] + [arrow left] / [arrow right]
  • Jump between events
    [S] + [arrow up] / [arrow down]
  • Enter value in empty cell
    [F]
  • Edit value
    [F] + [arrows]
  • Delete value
    [E]
  • Delete whole event
    [E] + [arrow right]
  • Delete whole row
    [E] + [arrow right] + [arrow right]
  • Cut
    [X]
  • Copy
    [C]
  • Paste
    [V]
  • Merge
    [B]
  • Undo
    [Z]
  • Redo
    [Y]
  • Note kill
    [K]
  • Toggle grab mode
    [G]
  • Create loop (in grab mode)
    [arrows] to select loop region, [space] to play/pause
  • Clear loop
    [G] + [E]
  • Set pattern length
    [L]

GRAB MODE

  • Toggle
    [G]
  • Grab whole event
    [G] + [arrow right]

When in grab mode, all navigation works as normal. For reference

  • Navigate
    [arrows]
  • Jump to end
    [A] + [arrows]
  • Jump between patterns (pattern sequencer only)
    [S] + [arrow left] / [arrow right]
  • Jump between events (pattern sequencer only)
    [S] + [arrow up] / [arrow down]
